# 🎮 Tic-Tac-Toe AI

An unbeatable **Tic-Tac-Toe** game developed in **Python** using the **Minimax Algorithm**. The AI analyzes every possible game state and always chooses the optimal move, making it impossible to defeat.

---

## 📌 Features

- 🤖 Unbeatable AI using the Minimax Algorithm
- 👤 Human vs Computer gameplay
- 🎯 Intelligent move selection
- 🧠 Recursive game-tree search
- 🏆 Win, Lose and Draw detection
- 🔄 Restart game functionality
- 📖 Clean and modular Object-Oriented code
- 💻 Simple terminal-based interface

## 🎮 How to Play

The board positions are numbered as follows:

```
 1 | 2 | 3
---+---+---
 4 | 5 | 6
---+---+---
 7 | 8 | 9
```

- You play as **X**
- Computer plays as **O**
- Enter the position number to place your move.
- The AI automatically makes its move after yours.

---

## 🧠 Minimax Algorithm

The AI uses the **Minimax Algorithm**, a decision-making algorithm widely used in Game Theory and Artificial Intelligence.

The algorithm:

- Generates all possible future game states.
- Evaluates each outcome.
- Maximizes the AI's chances of winning.
- Minimizes the player's chances of winning.
- Guarantees the best possible move every turn.

As a result, the AI can never be defeated if implemented correctly.
